🌍 Rest Countries Project

This project is a simple application that displays country data and uses JSON Server to simulate an API.

⚡️ How to Run the Project

1. Install dependencies:



npm install

2. Start the JSON Server on port 3000:



npx json-server --watch ./db/db.json --port 3000

3. Run the main project on port 3001:



npm run dev
# Or if you need to set the port explicitly:
PORT=3001 npm run dev

The app will be available at http://localhost:3001.
